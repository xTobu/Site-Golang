## Development

**Dependencies**
 - Go
 - Gin
 - go-sql-driver/mysql
 - google.golang.org/appengine
 - dgrijalva/jwt-go
 
**Database**
 - MySQL
 - Cloud SQL

**Hosting**  
 - Google App Engine  ([Link](https://xtobu-site.appspot.com))
  
  
  
##  Setup

#### Dev at localhost:8080
```bash
go run ./main.go
```

#### Deploy using Google Cloud SDK
```bash
gcloud app deploy
```
