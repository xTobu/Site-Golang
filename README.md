

# Site
> 基於 Golang 的 Back-End  

**Using**
 - Go
 - Gin
 - go-sql-driver/mysql
 
**Database**
 - Cloud SQL

**Hosting**  
 - Google App Engine  ([Link](https://xtobu-site.appspot.com))



##  Setup

#### Dev
```bash
# serve at localhost:8080
go run ./main.go
```

#### Deploy using Google Cloud SDK
```bash
gcloud app deploy
```
